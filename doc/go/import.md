

# Import 依赖
## 删除包
```bash
go mod edit -droprequire=github.com/casbin/casbin/v2
```

## 删除包和替换包
```bash
go mod edit -replace github.com/casbin/casbin/v2=github.com/casbin/casbin/v3@latest
```
